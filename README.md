A simple real-time weather app that provides current weather information using data from an open-source weather API.

## Features

- Real-time weather information.
- Location-based weather data.
- User-friendly interface.


## Technologies Used

- HTML
- CSS
- JavaScript

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, etc.).
- An API key from the OpenWeather API. You can obtain one by signing up on their website.
